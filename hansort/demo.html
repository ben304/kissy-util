功能：汉字拼音排序工具
描述：提供 数组/列表 按拼音或 Unicode 编码排序。在不支持默认拼音排序的操作系统里会自动加载一个 50k+(压缩后) ~ 80k+ 的汉字列表

    API

    方法

    KISSY.util.hanSort (array, config)

    参数

    • {Array} [ array = [] ] 需要排序的数组、NodeList、HTMLOListElement、HTMLUListElement、HTMLSelectElement，或者其选择器(String)
    • {Object} [ config = {}] 默认配置： 
      defaultCfg = {
    orderBy: "Pinyin",      // 0或"System"，系统默认排序；1或"Unicode"，笔画排序；2或"Pinyin"，拼音排序
    sort: 0                 // 0或"AES"，是升序；1或"DESC"，是降序
},

    返回

    排序后的数组
